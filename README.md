## Description
**Fresh Roots Recipes** is a free, user-friendly recipe website featuring hundreds of delicious dishes from around the world. The site contains five main pages: **Home**, **Categories**, **Ingredients**, **Country Cuisine**, and **Contact Us**.

* On the **Home** page, a random dish is shown each time the page loads.
* The **Categories** page lets users browse recipes by types such as Pasta, Dessert, Breakfast, and more.
* The **Ingredients** page allows users to select an ingredient and instantly view meals that use it.
* On the **Country Cuisine** page, users can filter dishes by country using the list displayed on the left side.
* Every page includes a site-wide **search bar** at the top for quick recipe lookup.
  
On the Home page, users can also search for a favorite meal, view matching results below the random dish, and click any meal to open a detailed recipe view‚Äîcomplete with ingredients, measurements, instructions, and a video tutorial.

## How to Use
Using the website is simple and intuitive.

* **Search Recipes:**
   Use the search bar on the Home page (or at the top of any page) to instantly see matching recipes.
* **Filter by Category, Ingredient, or Country:**
  Navigate to the respective pages to narrow recipes based on what you like.
* **View Recipe Details:**
  Click on any meal to open a recipe window showing the full recipe, including ingredients and a step-by-step video.
* **Find Meals Faster:**
  Use the top search bar on any page to quickly filter meals from large lists.
* **Watch Video Tutorials:**
  Each recipe pop-up includes a video link for users who prefer learning by watching.
