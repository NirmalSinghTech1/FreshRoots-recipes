## Description
`Fresh Roots Recipes` is a free, user-friendly recipe site with hundreds of recipes. This is five pages website - **Home**, **Categories**, **Ingredients**, **Country Cuisine** and **Contact Us**. On home page, user will see random dish every time page loads.
In Category page, There are many categories available such as Pasta, Dessert, Breakfast etc. User can select any prefered category and meals which comes under that category will display on the page. 
Ingredient page includes various types of ingredients, which will show list of meals includes that ingrient when select.  
In the County Cuisine page, there are many courtries list available on the left side, which can be used to filter cuisines based on the country. All pages have search field on the top, which user can use to search favouvite food. 
On the home page, user can search his/her favourite meal and it will display meals under the random meal. User can select the food to see the detailed recipe including ingredients with measures and a video link of the recipe. 

## How to Use
It is very easy to use this website. 
- On the home page, search for recipe in the search field. Matched recipes will display automatically.
- User can filter recipes by Categories, Ingredients and Countries.
- By clicking on meal, a recipe window will display which shows the detailed recipe.
- User can filter meal from many meals by using search field available on the top of the page.
- If user prefer video format then can see the video of the recipe, available under the image of the meal on the popup window.
